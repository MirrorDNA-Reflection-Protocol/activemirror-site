Operational Modes

LOCAL MODE (Air-Gapped)

  • ‚úì Zero outbound network calls
  • ‚úì Models run on your hardware (Ollama)
  • ‚úì Memory stored in local vault only
  • ‚úì Full audit log on filesystem
  • ‚úì Works offline
  • ‚úï Cannot use cloud models (Claude, GPT)

HYBRID MODE (Governed)

  • ‚úì Outbound calls ONLY to approved models
  • ‚úì Identity and memory stay local
  • ‚úì Governance layer runs locally
  • ‚úì Full audit log of all requests
  • ‚úì Works with Claude, GPT, Gemini
  • ‚úï Model providers see prompts (ToS applies)

You choose the trade-off. Both modes preserve local control of identity, memory, and governance.
