Core Layers

MirrorDNA Protocol

The foundational layer enabling identity persistence across sessions. Implements Structured Contextual Distillation (SCD) for memory without model modification.

Key files: ~/.mirrordna/, bus/, handoff.json

Memory Palace

Hierarchical memory system with hot (session), warm (recent), and cold (archival) layers. Implements semantic indexing via beacon glyphs.

Key components: memory.db, chromadb/, RAG server

MirrorGate Governance

Safety and governance layer with explicit constraints. All actions pass through governance checks before execution.

Key files: intent.md, SAFETY_CONTRACT.md

Inference Consortium

Multi-provider inference routing with local fallback. Routes queries to optimal provider based on task type, cost, and availability.

Providers: Groq, DeepSeek, Mistral, OpenAI, Ollama (local)

Memory Bus Protocol

The Memory Bus is the single source of truth for system state. All agents read from and write to the bus.

# Read current state
bus read

# Write after session
bus write "Session summary" --json '{"phase": "done"}'

# Check health
bus health
