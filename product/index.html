What Active Mirror Does

Active Mirror is a full-stack sovereign AI system. Here's what each layer provides.

Security Layer

AMGL Guard v1.1

Pre-inference security wrapper. Blocks prompt exfiltration, role injection, and social engineering before the model ever sees the request. Fail-closed — errors block, not allow.

Request Gate

Intent classification before inference. Requests categorized and routed based on type and risk.

Drift Monitor

Continuous SHA256 verification on identity files. Automatic alerts on any mismatch.

Identity Layer

AMI Kernel v1.0

Portable identity that works across Claude, Gemini, GPT, Ollama, and any MCP-compatible model. Your AI knows who you are — everywhere.
